# my-first-web

 This is my first web development project.

 I built a simple **self-introduction website** using HTML, CSS, and JavaScript to practice the basics of front-end development.

 ## Overview
 The goal of this project is to understand how HTML, CSS, and JavaScript work together to create a web page from scratch.

 ## Features
 - Self-introduction page
 - Basic layout and styling with CSS
 - Simple JavaScript interaction
 - Dark mode toggle functionality

## Technologies Used
 - HTML
 - CSS
 - JavaScript

 ## What I Learned 
 - How to structure a web page using HTML
 - How to style components with CSS
 - How to add basic interactivity using JavaScript
 - Basic project structure for web development

 ## Future Improvements

 - Add more JavaScript interactions
 - Make the website responsive for mobile devices
